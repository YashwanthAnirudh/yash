Process

  • Step 1: Data Discovery – Imported the dump.sql file(where the entire data is present) into SQL Workbench to set up the database for analysis, ensuring that all data in tables and views was accessible and ready for exploration.
  • Step 2: Data Analysis and Cleaning – Analyzed the database structure and standardized the inconsistent currency values in the transactions table by creating a new column named normalized_currency to convert all values to INR for consistency.
  • Step 3: Data Integration and ETL – Established a connection between the SQL database and Tableau using an ETL pipeline for seamless data access, ensuring that the cleaned data was dynamically available for visualization development in Tableau.

Key Insights

  • Created an interactive Tableau dashboard to analyze the sales trends of AtliQ hardware goods, enabling better decision-making for the sales team.
  • Surat led with the highest profit margin percentage at 4.86%, whereas Bengaluru showed a negative profit margin of -20.78%, indicating potential areas for operational improvement.
  • Total revenue over the period amounted to ₹984.87M, with a steady contribution from both e-commerce (24.4%) and brick-and-mortar (75.6%) channels.
  • Electrical stores emerged as the top customer segment, contributing ₹413.33M to revenue, while Prod040 was the best-performing product with sales of ₹23.58M.
